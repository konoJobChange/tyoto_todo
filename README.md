# tyoto_todo

ちょっとすごいToDo

## セットアップ

いっぺんにセットアップ

```console
make setup
```

フロントセットアップ

```console
make front-setup
```

## 実行

front

```console
make front-start
```

server-dev(ホットリロード付き)

```console
make server-dev
```

database

```console
make db-start
```

databaseへのコネクション(mysqlコマンドのインストール必要)

```console
make db-connect_local
```
